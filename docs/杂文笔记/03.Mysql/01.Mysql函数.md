# Mysql中实用函数

### COALESCE函数
::: note 语法
**COALESCE(value1,value2,...,...)**
:::
COALESCE函数可以有多个参数,并返回第一个非NULL参数。如果所有参数都为NULL，则COALESCE函数返回NULL。
**例子**
```mysql
select coalesce(NULL.NULL,NULL) from test   //返回NULL
select coalesce(name,1) from test           //name字段若是存在NULL值替换为1
```
